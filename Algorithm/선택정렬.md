### 선택 정렬 시간 복잡도

- N번 만큼 가장 작은 수를 찾아서 맨 앞으로 보내야 합니다.
- 구현 방식에 따라서 사소한 오차는 있을 수 있지만, 보통 전체 연산 횟순느
- N + (N+1) + (N+2) + .... + 2 입니다
- 이는 (N^2 + N - 2) / 2 로 표현할 수 있는데, 빅오 표기법에 따라서 O(N^2)이라고 작성합니다.
